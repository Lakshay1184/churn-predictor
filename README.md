# 💳 Customer Churn Prediction App

A machine learning-powered web application built using **Streamlit** that predicts whether a customer is likely to churn based on their banking details.

---

## 🚀 Project Overview

Customer churn is a critical problem in the banking sector. This project uses a trained deep learning model to analyze customer data and predict the probability of churn.

The app provides an intuitive UI where users can input customer details and get instant predictions.

---

## 🧠 Features

- Real-time churn prediction
- Clean and interactive UI (Streamlit)
- Encoded categorical inputs (Geography, Gender)
- Scaled numerical features
- Probability-based output with visual indicator

---

## 🛠️ Tech Stack

- Python
- Streamlit
- TensorFlow / Keras
- NumPy
- Scikit-learn
- Pickle / JSON
